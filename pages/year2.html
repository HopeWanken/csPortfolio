Cauldron

This project was created using p5js. It automatically stirrs, and begins to bubble when clicked. Clicking again stops the boil. When the bubbles resart in the cauldron, the color changes. This project is scalable, so by changing the variable cnSize, the whole scene shrinks or grows accordingly. A challenge was getting the colors to change every time the bubbles resarted. I fixed this by making the motion variable private to the function and creating a boolean that indicates wether the color should change.

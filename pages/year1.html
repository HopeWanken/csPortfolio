Mad Libs

This project was created using p5js. It allows you to input parts of speech and puts them into a story. The code randomly generates one of three stories. Input your words into the pop-ups using the appropriate part of speech. The story will automatically generate. To try it for yourself, click on the link below and follow the pop-ups.
This project was challenging because I had to learn how to allow the user to input values, and prevent the computer from running an infinite loop. I solved this problem by using the prompt() and noLoop() functions.

Bounce

This project was made in collaboration with Erin DeWitt using P5js. The lava bubbles have randomly generated positions. The black rectangle moves with the mouse. When the purple ball touches the rectangle, it bounces off. Each time it bounces, the score and speed increase. If the ball hits the lava, the game is over. To play again click on the link below.
A challenge in this project was creating the bounce() function. We struggled to figure out the correct directions (+/-) and if statement conditions. Once we figured it out, It made sense.
